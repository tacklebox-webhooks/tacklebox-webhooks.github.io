Overview

What Is It?

Tacklebox is a free, open-source serverless framework that offers webhooks as a service. Tacklebox as a project includes a CLI tool, a management UI, 4 client libraries (JavaScript, Ruby, Python and Go) a webhooks service which comprises: a RESTful API and AWS infrastructure.

 

Easy Setup

Deploy your own webhooks as a service infrastructure and dashboard using tacklebox deploy and tacklebox ui.
